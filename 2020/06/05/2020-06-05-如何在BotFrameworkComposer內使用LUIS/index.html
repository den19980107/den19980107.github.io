Posted on 2020-06-05 Edited on 2020-10-05

如何在 bot composer 內使用 Luis

  1. 前往 Luis 官網，使用微軟帳號登入

  2. 點選 New app for conversation

  3. 輸入 Name Culture，範例使用 English

  4. 進入剛剛創立好的 app 然後根據圖中指示把 Primary Key 記下來之後會用到

  5. 在 Bot composer 中建立一個 LUIS 範例 app Todo with LUIS

  6. 進入 bot 編輯頁面後點擊右上角的 start bot 來開始機器人

  7. 將剛剛紀錄的 Primary Key 填入，如果沒有 Primary Key 是不能使用這個功能的！

  8. 填入之後應該就可以啟用了
